# <GYM FX> 🏋️‍♂️

Un'applicazione desktop sviluppata in **JavaFX** che funge da guida visiva interattiva per gli esercizi da palestra.

L'app cataloga decine di esercizi suddividendoli in modo intuitivo per fasce muscolari (Pettorali, Dorsali, Gambe,
Spalle, Bicipiti, ecc.), fornendo per ciascuno un'immagine illustrativa chiara per capirne la corretta esecuzione.

## ✨ Funzionalità

* **Catalogo Completo:** Un'ampia libreria di esercizi suddivisi in categorie muscolari.
* **Supporto Visivo:** Immagini di alta qualità per ogni singolo esercizio.
* **Interfaccia Intuitiva:** Navigazione semplice tra le categorie grazie a un'interfaccia grafica pulita costruita con
  JavaFX (`.fxml` e `.css` personalizzati).

## 🛠️ Tecnologie Utilizzate

* **Linguaggio:** Java
* **Interfaccia Grafica:** JavaFX (FXML + CSS)
* **Architettura:** MVC (Model-View-Controller)
